This repository contains examples that illustrate how to use Vindral SDKs.

## Vindral Web SDK

Web SDK examples:

- Basic
- Channel Switching
- Embed
- Embed Responsive
- Audio-only
- Player (bundled)
- Custom Player

## Requirements

Latest stable nodejs and NPM version.

## Getting started

Install packages

`npm install`

Run

`npm run start`

Which will open the page in the browser at port 8080 or another free port.
